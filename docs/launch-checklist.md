# Launch Checklist

- Confirm canonical domain and redirect strategy.
- Set `NEXT_PUBLIC_SITE_URL`.
- Confirm address, phone, WhatsApp and email.
- Replace placeholder map URL.
- Review English and Arabic copy.
- Review privacy, cookies, terms and accessibility pages.
- Confirm all optional trust claims before enabling feature flags.
- Test all lead forms.
- Configure Odoo or generic lead webhook.
- Configure analytics only after cookie decision.
- Run lint, type-check, tests and production build.
- Submit sitemap to Search Console.
- Review mobile sticky CTA behaviour.
- Verify Google Business Profile consistency.
