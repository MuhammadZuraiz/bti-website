# Image credits & licensing

This file tracks the source and licence of every photo shipped in
`public/images/`. Stock photos are **placeholders** for authentic BTI
photography — because the image slots are path-based (see
[`src/config/media.ts`](../src/config/media.ts)), each can be swapped for a real
BTI photo later by dropping a file at the same path, with no code change.
