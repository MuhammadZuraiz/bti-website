# Launch Checklist

## Discovery and Ownership

- [ ] Read and archive the BTI readiness report.
- [ ] Obtain latest SEDD trade licence copy.
- [ ] Obtain VAT certificate or written VAT status confirmation.
- [ ] Obtain corporate-tax registration evidence if applicable.
- [ ] Confirm authorised signatory and production approver.
- [ ] Confirm canonical domain.
- [ ] Confirm registrar, DNS, hosting and email owners.
- [ ] Confirm social account owners.
- [ ] Confirm admissions, finance and content owners.

## Website Content

- [ ] Approve sitemap.
- [ ] Approve navigation labels in English and Arabic.
- [ ] Approve course categories.
- [ ] Approve final course names.
- [ ] Approve course descriptions.
- [ ] Approve schedules and intake wording.
- [ ] Approve fees or remove fees from public pages.
- [ ] Approve official IELTS/test-centre wording.
- [ ] Approve contact details and address format.
- [ ] Approve privacy, terms, cookies and accessibility pages.

## Technical Setup

- [ ] Configure canonical domain.
- [ ] Configure redirects from legacy domains.
- [ ] Enable HTTPS.
- [ ] Configure Cloudflare DNS/CDN/WAF.
- [ ] Configure SPF, DKIM and DMARC.
- [ ] Configure analytics.
- [ ] Configure Search Console.
- [ ] Configure uptime monitoring.
- [ ] Configure 404 monitoring.
- [ ] Configure backups and restore test.

## Odoo

- [ ] Create Odoo.sh project.
- [ ] Configure dev, staging and production branches.
- [ ] Configure company details only after licence verification.
- [ ] Create user groups and named users.
- [ ] Configure Contacts, CRM, Sales, Invoicing and Website.
- [ ] Add lead fields.
- [ ] Configure CRM stages.
- [ ] Create course products.
- [ ] Configure tax rules after finance review.
- [ ] Test lead to quote.
- [ ] Test quote to invoice.
- [ ] Test payment receipt.
- [ ] Test enrolment/cohort assignment.

## Testing

- [ ] Desktop Chrome/Edge test.
- [ ] Mobile viewport test.
- [ ] Keyboard-only navigation test.
- [ ] Form required-field test.
- [ ] Consent checkbox test.
- [ ] Arabic RTL layout test.
- [ ] Course filters test.
- [ ] Legal page links test.
- [ ] Sitemap and robots check.
- [ ] Structured data validation.
- [ ] Lighthouse or equivalent performance check.

## Go-Live

- [ ] Freeze content.
- [ ] Backup old website assets.
- [ ] Confirm rollback owner.
- [ ] Update DNS.
- [ ] Verify SSL.
- [ ] Verify redirects.
- [ ] Submit sitemap.
- [ ] Submit test lead.
- [ ] Confirm lead appears in CRM.
- [ ] Confirm staff receive notification.
- [ ] Check analytics real-time visit.
- [ ] Check mobile CTA links.

## Post-Launch

- [ ] Review leads daily for first week.
- [ ] Review 404s daily for first week.
- [ ] Review Search Console after indexing.
- [ ] Run weekly form checks.
- [ ] Run monthly patch review.
- [ ] Run quarterly public listing audit.
- [ ] Run quarterly content owner review.
- [ ] Run backup restore test monthly.
