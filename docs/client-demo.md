## Current-state risk -> Website improvement -> Business value

- Broken or fragmented web presence -> One polished canonical site -> More trust
- Scattered enquiries -> Structured lead capture -> Better follow-up
- Generic contact-only experience -> Course-specific pages and CTAs -> More qualified leads
- Weak search visibility -> Local SEO pages and metadata -> Better discoverability
- Mobile friction -> Mobile-first experience and sticky CTAs -> Easier enquiries
- No attribution -> Analytics events and UTM capture -> Better marketing decisions
